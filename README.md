# Housing Price Prediction

California housing price prediction with feature engineering.

## 📋 Requirements

- Python 3.7+
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn (optional)

## 🚀 Quick Start

```bash
python Housing.py
```

Or run the Jupyter notebook:
```bash
jupyter notebook Housing.ipynb
```
